## How to use this repo

1. Clone this repository and `cd` into the local directory.

2. Use the Node.js version specified in `.nvmrc`.

    ```
    nvm use
    ```

2. Install all dependencies.

    ```
    npm install
    ```

3. Get a list of available commands.

    ```
    npm run
    ```
